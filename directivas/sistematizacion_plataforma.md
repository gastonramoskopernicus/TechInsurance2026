# Directiva: Sistemática de Diseño Premium - Página Plataforma

## Objetivos
1. **Unificación Visual con Home:** Migrar toda la página `/plataforma` al ecosistema Magenta/Zinc (`bg-[#0f0e11]`, `bg-[#130a16]`). Aplicar Text Gradients a los focos semánticos de todos los encabezados `<h2>` para mantener continuidad tipográfica.
2. **Rejerarquización de Product Studio:** Ubicar este bloque como el núcleo central (justo post-Capacidades), insertando el imagotipo vectorial de la marca respirando gravitacionalmente en su cúspide y aumentando dramáticamente sus márgenes de respiro.
3. **Conversión Estructural de Contenidos:** 
   - Transmutar viñetas aburridas de "Capacidades Centrales" a `Cards de UI Activas` (iconos vectoriales massive y hover dinámico).
   - Elevar los "Módulos Core" de listados simples a un Grid de tarjetas modulares.
   - Otorgar Identidad Geométrica a cada sección suplementaria (Arquitectura, API, Seguridad, Orquestación) utilizando cajas con bordes y brillos fuchsia.
4. **Refinamiento UI de Navbar Interno:** Transicionar el submenú sticky flotante a una tonalidad borgoña sutil (`bg-gradient-to-r from-[#200e23]/90 to-black/90` con bordes magenta translúcidos).

## Restricciones Operativas
- **No reescribir** ninguna de las promesas de valor (textos). Todos los copys se embeben crudos en la nueva matriz de HTML.
- **Respetar estrictamente** todos los atributos `id=` del menú interno para garantizar que el anclaje de scroll (ScrollSpy) nunca se rompa.
